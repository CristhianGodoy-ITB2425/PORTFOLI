Docker

En este proyecto, diseñé y desarrollé una infraestructura moderna utilizando Docker, una herramienta que facilita la creación y gestión de aplicaciones dentro de contenedores aislados. Esta tecnología me permitió crear un entorno eficiente, seguro y escalable para implementar diferentes servicios esenciales.

Configuré un contenedor con Apache para alojar y servir páginas web de manera rápida, confiable y segura.

Implementé un contenedor que incluye phpMyAdmin, una herramienta gráfica que facilita la administración de bases de datos MySQL o MariaDB a través del navegador.

Diseñé un contenedor especializado con herramientas anti-rootkit para detectar y eliminar posibles amenazas que pudieran comprometer el sistema.

Configuré un contenedor dedicado que actúa como un firewall, encargado de filtrar y controlar el tráfico de red para proteger toda la infraestructura.


¿Por qué utilicé Docker?

Docker fue la pieza clave para este proyecto porque me permitió: Empaquetar cada servicio en un contenedor independiente, lo que asegura portabilidad y compatibilidad en cualquier sistema operativo.

Mejorar el rendimiento al usar contenedores más ligeros que las máquinas virtuales.

Garantizar aislamiento entre servicios, evitando conflictos y facilitando el mantenimiento.

Implementar una infraestructura modular y escalable, lista para adaptarse a las necesidades de cualquier entorno.
